## C++ Userã€€<a id="cpp"/>

### Prerequisite
* OS: ubuntu 22.04 **with sudo privilege**
* Development environment
    * C++
* Hardware:
    * USB3.0 compatible USB port x2

### Download the setup script

<pre>
<code class="language-bash">
wget -O setup.sh {this_version.latest_setup_URL}
</code>
</pre>

### Install and setup SDK

Note that `--install-opencv`, `--install-gst-tools`, and `--install-gst-plugin` are optional.

The following command install SDK and its dependencies with optional contents.

<pre><code class="language-bash">sudo bash setup.sh --install-opencv --install-gst-tools --install-gst-plugin {this_version.linux_version}
</code></pre>

* `--install-opencv`: Add this option to install OpenCV with Sensing-Dev SDK.
* `--install-gst-tools`: Add this option to install Gstreamer tools with Sensing-Dev SDK.
* `--install-gst-plugin`: Add this option to install Gstreamer plugins with Sensing-Dev SDK.

### Set Environment Variables

<pre>
<code class="language-bash">
export LD_LIBRARY_PATH=/opt/sensing-dev/lib:/opt/sensing-dev/lib/x86_64-linux-gnu:$LD_LIBRARY_PATH
</code>
</pre>

:::info Confirmation
Using `/opt/sensing-dev/bin/arv-tool-0.8` may help to check if the software package is appropriately installed. Please check [this page](../external/aravis/arv-tools) to know the procedure.
:::

### Uninstall SDK

When you want to uninstall the package introduced by Sensing-Dev, please execute the following command.

```
sudo rm -r /opt/sensing-dev
```

## Python User <a id="python"/>
### Prerequisite
* OS: ubuntu 22.04 **with sudo privilege**
* Development environment
    * Python >=3.10
* Hardware:
    * USB3.0 compatible USB port x2

:::info For users of previous versions
Sensing-Dev v24.09 or later uses ion-contrib-python instead of ion-python. Therefore, please uninstall ion-python with the following command.

<pre>
<code class="language-powershell">
pip uninstall ion-python -y
</code>
</pre>
:::

### Install dependency
<pre>
<code class="language-bash">
sudo apt install -y libgirepository1.0-dev gcc libcairo2-dev pkg-config python3-dev gir1.2-gtk-4.0<br />
pip3 install -U pip<br />
pip3 install pycairo<br />
</code>
</pre>

### Install Python Modules

Install aravis-python and gendc-python and ion-contrib-python to fully utilize. It is optional to install opencv-python, which is required for some of tutorials.

<pre>
<code class="language-bash">
pip3 install PyGObject<br />
pip3 install aravis-python=={this_version.aravis_python_version}<br />
pip3 install ion-contrib-python=={this_version.ion_python_version}<br />
pip3 install gendc-python=={this_version.gendc_python_version}<br />
pip3 install opencv-python<br />
</code>
</pre>

### Uninstall Python Modules

When you want to uninstall the modules introduced by Sensing-Dev, please execute the following command.

<pre>
<code class="language-bash">
pip3 uninstall aravis-python ion-contrib-python gendc-python PyGObject  -y
</code>
</pre>
